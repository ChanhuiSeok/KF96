<h3 align="center"> 현재 위치와 공공데이터를 이용하여 미세먼지 정보 제공과 캘린더 저장 및 시각화</h3>

# 📋 기능
✔ 　실시간 초미세먼지, 미세먼지 정보 제공    
✔ 　통합대기 정보 제공  
✔ 　10분을 주기로 미세먼지 노출량 캘린더에 저장  
✔ 　캘린더를 이용하여 사용자의 일일 미세먼지 노출량 정보 제공  
✔ 　사용자의 주간 미세먼지 노출 데이터 시각화
